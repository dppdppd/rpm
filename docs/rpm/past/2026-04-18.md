### Key Discoveries
- **opencode loads plugins lazily on first `POST /session`.** The
  first session's `session.created` event publishes BEFORE the
  plugin's event hook registers, so the first session misses it.
  Workaround: run SessionStart logic inline inside the Plugin init
  function (which runs once per project bootstrap); subsequent
  `session.created` events are caught by the event hook.
- **opencode Zen free models** (`opencode/minimax-m2.5-free`,
  `opencode/nemotron-3-super-free`) require no credentials — enable
  interactive TUI/run testing without an API key. Unblocked
  distribution validation.
- **opencode's `Hooks` interface has no dedicated session-lifecycle
  hooks.** Lifecycle events arrive via a single generic `event` hook
  receiving a discriminated-union `Event` object; dispatch is a
  `switch (event.type)`. Only `experimental.session.compacting` and
  `tool.execute.after` are dedicated.
- **Commands vs skills surface mismatch.** opencode skills are
  tool-loaded context (no `!bash` / `$ARGUMENTS` expansion);
  commands are slash-invocable with the same template syntax as
  Claude Code. rpm's six entry points belong in `.opencode/commands/`,
  NOT `.opencode/skills/`. Mirroring skills alone makes them
  discoverable but non-invocable.
- **`fileURLToPath(import.meta.url)` preserves symlinks.** Need
  `realpathSync` to unfurl — otherwise relative-path resolution from
  a symlinked plugin lands in the wrong place (confirmed by a dev
  install via symlink into `/tmp/rpm-oc-test/.opencode/plugins/`
  that produced `hooks=/tmp/plugin/hooks`).
- **`git add -f opencode/` sweeps gitignored cruft.** First publish
  attempt split 3552 files because `node_modules/` was force-added.
  Fix: force-add specific mirror paths only.
- **opencode agent frontmatter is incompatible with Claude Code's.**
  `tools: [...]` (array) is rejected as "expected record, received
  array"; needs `tools: {read: true, ...}` with lowercase tool
  names. Also: `model` shortnames (`sonnet`) don't map to opencode's
  provider-prefixed format (`anthropic/claude-sonnet-4-...`).
